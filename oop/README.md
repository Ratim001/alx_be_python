This directory explores the principles of Object-Oriented Programming (OOP) in Python through practical demonstrations. - **Book Class**:

  - Explains the usage of `@classmethod` and `@staticmethod` decorators for flexible function calls.
- **Library System**:
  - Implements inheritance to manage various types of books (e.g., `EBook`, `PrintBook`) in a library.
- **Polymorphism Demo**:
  - Showcases polymorphic behavior by calculating areas of different shapes using shared method names.
